# Web bán hàng hiện đại
> Website thương mại điện tử UI hiện đại, responsive, sử dụng Next.js 15, TailwindCSS v3 và nhiều thư viện tiện ích khác.

---

##  Hướng dẫn cài đặt FE

```bash
cd client
npm install
npm run dev
```

---

##  Công nghệ sử dụng

### Framework
- **Next.JS v15**

### Thư viện
- `react-multi-carousel`
- `react-icons`
- `react-toastify`
- `tailwindcss` v3

---

##  Hướng dẫn cài đặt BE

```bash
cd server
npm install
npm run dev
```

##  Công nghệ sử dụng

### Framework
- **Node & Express.JS**

### Thư viện
- `cors`
- `dotenv`
- `mongoose`
- `morgan`
