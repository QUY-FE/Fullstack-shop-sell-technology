# Web bán hàng hiện đại
> Website thương mại điện tử UI hiện đại, responsive, sử dụng Next.js 15, TailwindCSS v3 và nhiều thư viện tiện ích khác.

---

##  Hướng dẫn cài đặt

```bash
npm install
npm run dev
```
